第一章 系统概述

1.1 系统简介
本系统为公司自主研发的企业资源规划平台，集成了采购管理、库存管理、销售管理、财务管理及人力资源管理五大核心模块。系统采用B/S架构，支持Chrome、Firefox、Edge等主流浏览器，推荐使用Chrome 110及以上版本以获得最佳体验。

1.2 系统访问地址
- 内网访问：http://erp.internal.company.com
- 外网访问（需VPN）：https://erp.company.com
- 移动端：下载"XX企业云"APP，在设置中配置企业服务器地址

1.3 账号权限说明
系统采用角色权限控制（RBAC），共设以下角色：
- 普通员工：可查看本部门相关数据，不可修改
- 部门主管：可查看和修改本部门数据，可审批本部门流程
- 财务人员：可操作财务模块全部功能
- 系统管理员：拥有全部权限，负责用户管理和系统配置
新员工账号由IT部门统一创建，默认密码为"工号@Company"，首次登录须强制修改密码。

第二章 采购管理模块

2.1 采购申请流程
步骤一：登录系统后，点击顶部导航"采购管理" → "采购申请" → "新增申请"
步骤二：填写采购申请单，必填字段包括：
  - 申请部门（自动带入）
  - 物料名称及规格型号
  - 申请数量及单位
  - 预计单价（参考历史采购价）
  - 需求日期（须距申请日至少5个工作日）
  - 用途说明（不少于50字）
步骤三：上传相关附件（如供应商报价单、技术规格书等）
步骤四：提交申请，系统自动推送审批流程

2.2 采购申请审批规则
根据采购金额，审批层级自动调整：
- 5000元以下：部门主管审批 → 财务确认 → 采购执行
- 5000-50000元：部门主管 → 部门总监 → 财务审批 → 采购执行
- 50000元以上：部门主管 → 部门总监 → 财务审批 → 总经理审批 → 采购执行
审批超时规则：每个审批节点超过48小时未处理，系统自动发送催办通知；超过72小时未处理，自动升级通知上级。

2.3 常见问题：采购申请提交失败
错误提示"供应商信息不完整"：请确认选择的供应商已在系统中完成资质录入，如供应商为新供应商，须先由采购部完成供应商准入审核。
错误提示"预算不足"：当前部门可用预算余额不足，请联系财务部申请预算调整。

第三章 库存管理模块

3.1 入库操作
当采购订单状态变更为"已发货"后，仓库人员可进行入库登记：
路径：库存管理 → 入库管理 → 收货确认
操作要点：
- 实收数量可与采购数量不一致（如货物损坏、短缺），系统会自动生成差异记录
- 入库时必须扫描物料条码，不支持手工录入物料编码以防止错误
- 入库完成后，库存数量实时更新，相关采购订单状态变更为"已收货"

3.2 库存盘点
系统支持全盘和抽盘两种模式：
- 全盘：每年1月进行，盘点期间相关仓库冻结，不可进行入出库操作
- 抽盘：各部门可随时发起，不影响正常业务

盘点操作步骤：
1. 发起盘点任务（库存管理 → 盘点管理 → 新建盘点）
2. 打印盘点清单，按清单逐一核实实物
3. 在系统中录入实盘数量
4. 提交盘点结果，财务确认后系统自动调整库存数量

3.3 库存预警设置
管理员可为每种物料设置库存预警线：
- 最低库存：低于该值时系统发送补货提醒
- 最高库存：超过该值时系统发出过量库存警告
路径：库存管理 → 物料设置 → 预警配置

第四章 财务管理模块

4.1 费用报销流程
步骤一：员工在系统中新建报销单（财务管理 → 费用报销 → 新增报销）
步骤二：选择费用类型（交通费、餐费、住宿费、办公用品等），上传票据照片
步骤三：提交审批，审批通过后财务完成打款
报销规定：
- 餐费标准：本地出差100元/天，外地出差150元/天
- 住宿标准：一线城市500元/晚，二线城市350元/晚，其他城市250元/晚
- 交通费：飞机仅限总监及以上级别或特殊情况申请，需附申请说明

4.2 常见报销问题
Q：发票抬头应该填什么？
A：发票抬头须填写公司全称"XX科技有限公司"，税号为91110108XXXXXXXX，不符合规定的发票财务不予报销。

Q：出差提前预订酒店，发票开具时间早于出差时间怎么办？
A：属于正常情况，报销时在备注栏注明"提前预订"并附行程证明即可。

第五章 系统故障处理

5.1 无法登录系统
可能原因及解决方法：
1. 密码错误：连续5次输入错误密码，账号将被锁定30分钟。请等待后重试，或联系IT帮台解锁。
2. 账号被禁用：员工离职或权限变更后账号会被禁用，如误操作请联系IT部门。
3. 浏览器问题：请清除浏览器缓存后重试，或更换Chrome浏览器。
4. 网络问题：内网访问请确认连接公司WiFi或有线网络；外网访问请确认已连接VPN。

5.2 系统运行缓慢
1. 检查浏览器是否打开过多标签页，建议关闭无关标签
2. 避免在报表导出等大数据操作期间进行其他操作
3. 如问题持续存在，请记录操作步骤和时间后联系IT帮台

5.3 数据导出功能
系统支持将查询结果导出为Excel或PDF格式：
- Excel导出：适合需要二次处理数据的场景，单次最多导出10万行
- PDF导出：适合存档或打印，保留原始格式
注意：报表数据导出须经部门主管授权，敏感财务数据导出须财务经理审批。

5.4 IT支持联系方式
- IT帮台热线：分机 8888（工作日 8:30-18:00）
- 紧急故障（生产系统中断）：值班手机 138-XXXX-XXXX（7×24小时）
- 邮件工单：it-support@company.com（非紧急问题，1个工作日内响应）
